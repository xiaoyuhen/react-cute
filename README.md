# Install 🚀
  ```
  $ yarn add react-cute
  ```

# Usage ⛏

```jsx
import Cute from 'react-cute';

const json = {
  name: 'hector',
  twitter: '@zzarcon'
};

<Cute json={json} />
```

# Features ✨
  * Out of the box UI
  * Different colors based on value type
  * Render values not showed by default
  * Dependency free + no external CSS
  * [3kB size](https://bundlephobia.com/result?p=react-cute)
